
### Step-by-Step: Setup Node.js + Expres
1. node -v
   npm -v

2. Create Project Folder
   mkdir fullstack-server
   cd fullstack-server

3. Initialize a Node Project
   npm init -y

4. Install Express   
   npm install express

 5. Create Your Server File  

     server.js

### to start the server
node server 

### or 

npx nodemon server.js
